Taller Guiado: Solución Paso a Paso de Ecuaciones con Radicales

Este taller presenta la solución detallada de ecuaciones con radicales, desarrollando cada paso del proceso y explicando los conceptos matemáticos aplicados. Cada ejercicio incluye:

  • Ecuación original - El planteamiento inicial del problema
  • Restricciones - Condiciones que debe cumplir la variable por la presencia de radicales
  • Proceso de resolución - Secuencia ordenada de transformaciones algebraicas
  • Verificación - Comprobación de las soluciones en la ecuación original

Ejercicio 1: \(x=4+\sqrt{4x-19}\)
Ecuación original:
\[ x=4+\sqrt{4x-19} \]
Restricción:

El radicando debe ser no negativo:

\[ 4x-19 \geq 0 \implies 4x \geq 19 \implies x \geq \frac{19}{4} = 4.75 \]
Concepto:

Aislar el término con la raíz cuadrada restando 4 de ambos lados.

\[ x - 4 = \sqrt{4x-19} \]
Restricción implícita:

El lado izquierdo debe ser no negativo para que sea igual a la raíz cuadrada principal:

\[ x-4 \geq 0 \implies x \geq 4 \]

Combinando las restricciones, debemos tener:

\[ x \geq \max\left(\frac{19}{4}, 4\right) = \frac{19}{4} = 4.75 \]
Concepto:

Elevar ambos lados al cuadrado para eliminar la raíz cuadrada.

\[ (x - 4)^2 = (\sqrt{4x-19})^2 \] \[ x^2 - 8x + 16 = 4x - 19 \]
Concepto:

Reorganizar la ecuación para formar una ecuación cuadrática.

\[ x^2 - 8x - 4x + 16 + 19 = 0 \] \[ x^2 - 12x + 35 = 0 \]
Concepto:

Factorizar el trinomio cuadrado. Buscamos dos números que multiplicados den 35 y sumados den -12. Los números son -5 y -7.

\[ (x - 5)(x - 7) = 0 \]
Concepto:

Establecer cada factor igual a cero y resolver para \(x\).

\[ x - 5 = 0 \implies x = 5 \] \[ x - 7 = 0 \implies x = 7 \]
Verificación:

Para \(x=5\):

  • Restricción \(5 \geq 4.75\) es verdadera.
  • Sustituyendo: \[ 5 = 4 + \sqrt{4(5)-19} = 4 + \sqrt{20-19} = 4 + \sqrt{1} = 4 + 1 = 5 \]
  • La solución es válida.

Para \(x=7\):

  • Restricción \(7 \geq 4.75\) es verdadera.
  • Sustituyendo: \[ 7 = 4 + \sqrt{4(7)-19} = 4 + \sqrt{28-19} = 4 + \sqrt{9} = 4 + 3 = 7 \]
  • La solución es válida.
Soluciones:
\[ x = 5, 7 \]

Ejercicio 2: \(\sqrt{7-2x}-\sqrt{5+x}=\sqrt{4+3x}\)
Ecuación original:
\[ \sqrt{7-2x}-\sqrt{5+x}=\sqrt{4+3x} \]
Restricciones:

Los radicandos deben ser no negativos:

\[ 7-2x \geq 0 \implies -2x \geq -7 \implies x \leq \frac{7}{2} = 3.5 \] \[ 5+x \geq 0 \implies x \geq -5 \] \[ 4+3x \geq 0 \implies 3x \geq -4 \implies x \geq -\frac{4}{3} \approx -1.33 \]

La restricción combinada es:

\[ -\frac{4}{3} \leq x \leq \frac{7}{2} \]
Concepto:

Elevar ambos lados al cuadrado. (Esto puede introducir soluciones extrañas).

\[ (\sqrt{7-2x}-\sqrt{5+x})^2 = (\sqrt{4+3x})^2 \] \[ (\sqrt{7-2x})^2 - 2\sqrt{7-2x}\sqrt{5+x} + (\sqrt{5+x})^2 = 4+3x \] \[ (7-2x) - 2\sqrt{(7-2x)(5+x)} + (5+x) = 4+3x \]
Concepto:

Simplificar la expresión.

\[ 7 - 2x - 2\sqrt{35 + 7x - 10x - 2x^2} + 5 + x = 4+3x \] \[ 12 - x - 2\sqrt{35 - 3x - 2x^2} = 4+3x \]
Concepto:

Aislar el término con la raíz cuadrada.

\[ -2\sqrt{35 - 3x - 2x^2} = 4 + 3x - 12 + x \] \[ -2\sqrt{35 - 3x - 2x^2} = 4x - 8 \]
Concepto:

Dividir ambos lados por -2.

\[ \sqrt{35 - 3x - 2x^2} = -2x + 4 \]
Restricción implícita:

El lado derecho debe ser no negativo:

\[ -2x+4 \geq 0 \implies -2x \geq -4 \implies x \leq 2 \]

Combinando con la restricción principal:

\[ -\frac{4}{3} \leq x \leq 2 \]
Concepto:

Elevar ambos lados al cuadrado nuevamente.

\[ (\sqrt{35 - 3x - 2x^2})^2 = (-2x + 4)^2 \] \[ 35 - 3x - 2x^2 = 4x^2 - 16x + 16 \]
Concepto:

Reorganizar la ecuación para formar una ecuación cuadrática.

\[ 0 = 4x^2 + 2x^2 - 16x + 3x + 16 - 35 \] \[ 0 = 6x^2 - 13x - 19 \]
Concepto:

Utilizar la fórmula general para resolver la ecuación cuadrática, donde \(a=6\), \(b=-13\), \(c=-19\).

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \] \[ x = \frac{-(-13) \pm \sqrt{(-13)^2 - 4(6)(-19)}}{2(6)} \] \[ x = \frac{13 \pm \sqrt{169 + 456}}{12} \] \[ x = \frac{13 \pm \sqrt{625}}{12} \] \[ x = \frac{13 \pm 25}{12} \]
Concepto:

Calcular las dos posibles soluciones.

\[ x_1 = \frac{13 + 25}{12} = \frac{38}{12} = \frac{19}{6} \approx 3.17 \] \[ x_2 = \frac{13 - 25}{12} = \frac{-12}{12} = -1 \]
Verificación:

Para \(x=\frac{19}{6} \approx 3.17\):

  • Restricción \(-\frac{4}{3} \leq \frac{19}{6} \leq 2\) es falsa porque \(\frac{19}{6} > 2\).
  • \(x=\frac{19}{6}\) es una solución extraña.

Para \(x=-1\):

  • Restricción \(-\frac{4}{3} \leq -1 \leq 2\) es verdadera.
  • Sustituyendo en el lado izquierdo: \[ \sqrt{7-2(-1)}-\sqrt{5+(-1)} = \sqrt{7+2}-\sqrt{4} = \sqrt{9}-2 = 3-2 = 1 \]
  • Sustituyendo en el lado derecho: \[ \sqrt{4+3(-1)} = \sqrt{4-3} = \sqrt{1} = 1 \]
  • \(1 = 1\). La solución es válida.
Solución:
\[ x = -1 \]

Ejercicio 3: \(\sqrt{2\sqrt{x+1}}=\sqrt{3x-5}\)
Ecuación original:
\[ \sqrt{2\sqrt{x+1}}=\sqrt{3x-5} \]
Restricciones:

Los radicandos deben ser no negativos:

\[ x+1 \geq 0 \implies x \geq -1 \] \[ 3x-5 \geq 0 \implies 3x \geq 5 \implies x \geq \frac{5}{3} \approx 1.67 \]

Además, el radicando bajo la primera raíz cuadrada debe ser no negativo:

\[ 2\sqrt{x+1} \geq 0 \]

Lo cual es cierto siempre que \(x+1 \geq 0\).

La restricción combinada es:

\[ x \geq \frac{5}{3} \]
Concepto:

Elevar ambos lados al cuadrado para eliminar las raíces cuadradas externas.

\[ (\sqrt{2\sqrt{x+1}})^2 = (\sqrt{3x-5})^2 \] \[ 2\sqrt{x+1} = 3x - 5 \]
Restricción implícita:

El lado derecho debe ser no negativo:

\[ 3x-5 \geq 0 \implies x \geq \frac{5}{3} \]

Esta es la misma restricción principal.

Concepto:

Elevar ambos lados al cuadrado nuevamente para eliminar la raíz cuadrada restante.

\[ (2\sqrt{x+1})^2 = (3x - 5)^2 \] \[ 4(x+1) = 9x^2 - 30x + 25 \] \[ 4x + 4 = 9x^2 - 30x + 25 \]
Concepto:

Reorganizar la ecuación para formar una ecuación cuadrática.

\[ 0 = 9x^2 - 30x - 4x + 25 - 4 \] \[ 0 = 9x^2 - 34x + 21 \]
Concepto:

Utilizar la fórmula general para resolver la ecuación cuadrática, donde \(a=9\), \(b=-34\), \(c=21\).

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \] \[ x = \frac{-(-34) \pm \sqrt{(-34)^2 - 4(9)(21)}}{2(9)} \] \[ x = \frac{34 \pm \sqrt{1156 - 756}}{18} \] \[ x = \frac{34 \pm \sqrt{400}}{18} \] \[ x = \frac{34 \pm 20}{18} \]
Concepto:

Calcular las dos posibles soluciones.

\[ x_1 = \frac{34 + 20}{18} = \frac{54}{18} = 3 \] \[ x_2 = \frac{34 - 20}{18} = \frac{14}{18} = \frac{7}{9} \approx 0.78 \]
Verificación:

Para \(x=3\):

  • Restricción \(3 \geq \frac{5}{3}\) es verdadera.
  • Sustituyendo en el lado izquierdo: \[ \sqrt{2\sqrt{3+1}} = \sqrt{2\sqrt{4}} = \sqrt{2 \times 2} = \sqrt{4} = 2 \]
  • Sustituyendo en el lado derecho: \[ \sqrt{3(3)-5} = \sqrt{9-5} = \sqrt{4} = 2 \]
  • \(2 = 2\). La solución es válida.

Para \(x=\frac{7}{9} \approx 0.78\):

  • Restricción \(\frac{7}{9} \geq \frac{5}{3}\) es falsa porque \(\frac{7}{9} \approx 0.78\) y \(\frac{5}{3} \approx 1.67\).
  • \(x=\frac{7}{9}\) es una solución extraña.
Solución:
\[ x = 3 \]

Ejercicio 4: \(\sqrt{1+4\sqrt{x}}=\sqrt{x}+1\)
Ecuación original:
\[ \sqrt{1+4\sqrt{x}}=\sqrt{x}+1 \]
Restricciones:

Los radicandos deben ser no negativos:

\[ x \geq 0 \] \[ 1+4\sqrt{x} \geq 0 \]

Lo cual es cierto siempre que \(x \geq 0\).

La restricción principal es:

\[ x \geq 0 \]
Concepto:

Elevar ambos lados al cuadrado.

\[ (\sqrt{1+4\sqrt{x}})^2 = (\sqrt{x}+1)^2 \] \[ 1+4\sqrt{x} = (\sqrt{x})^2 + 2(\sqrt{x})(1) + 1^2 \] \[ 1+4\sqrt{x} = x + 2\sqrt{x} + 1 \]
Concepto:

Aislar el término con la raíz cuadrada en un lado.

\[ 4\sqrt{x} - 2\sqrt{x} = x + 1 - 1 \] \[ 2\sqrt{x} = x \]
Concepto:

Elevar ambos lados al cuadrado nuevamente.

\[ (2\sqrt{x})^2 = x^2 \] \[ 4x = x^2 \]
Concepto:

Reorganizar la ecuación y factorizar para resolver la ecuación cuadrática.

\[ 0 = x^2 - 4x \] \[ 0 = x(x - 4) \]
Concepto:

Establecer cada factor igual a cero y resolver para \(x\).

\[ x = 0 \] \[ x - 4 = 0 \implies x = 4 \]
Verificación:

Para \(x=0\):

  • Restricción \(0 \geq 0\) es verdadera.
  • Sustituyendo en el lado izquierdo: \[ \sqrt{1+4\sqrt{0}} = \sqrt{1+4(0)} = \sqrt{1} = 1 \]
  • Sustituyendo en el lado derecho: \[ \sqrt{0}+1 = 0+1 = 1 \]
  • \(1 = 1\). La solución es válida.

Para \(x=4\):

  • Restricción \(4 \geq 0\) es verdadera.
  • Sustituyendo en el lado izquierdo: \[ \sqrt{1+4\sqrt{4}} = \sqrt{1+4(2)} = \sqrt{1+8} = \sqrt{9} = 3 \]
  • Sustituyendo en el lado derecho: \[ \sqrt{4}+1 = 2+1 = 3 \]
  • \(3 = 3\). La solución es válida.
Soluciones:
\[ x = 0, 4 \]

Conclusión y Recomendaciones

La solución de ecuaciones con radicales requiere un análisis cuidadoso que incluye:

  1. Establecer restricciones para los dominios de las funciones radicales
  2. Elevar al cuadrado para eliminar los radicales (prestando atención a posibles soluciones extrañas)
  3. Resolver las ecuaciones resultantes aplicando técnicas algebraicas adecuadas
  4. Verificar todas las soluciones en la ecuación original para descartar soluciones extrañas

Las soluciones extrañas suelen aparecer porque al elevar ambos lados de una ecuación al cuadrado se pierde la información sobre el signo. Por eso, la verificación en la ecuación original es un paso esencial en el proceso de resolución.
